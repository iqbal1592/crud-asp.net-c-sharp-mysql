# ASP.Net Sample Application with CRUD Operations for MySQL Database
It is an ASP.Net sample application that performs CRUD (Create, Read, Update and Delete) operations for MySQL database. It uses C# as the programming language.

<h3>Step 1:</h3>
Run the script in &lt;root&gt;/Database/<strong>HRMS.sql</strong> file to create the MySQL database.

<h3>Step 2:</h3>
Change the connection string in the <strong>Web.config</strong> file.

<h3>Step 3:</h3>
Run the application using Visual Studio or deploy it on IIS.
